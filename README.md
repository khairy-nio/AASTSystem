## Routes

| Route | Description |
|---|---|
| `/` | Landing page — overview of system features and entry point to login/register |
| `/login` | Sign in using Employee ID and password |
| `/register` | Register a new account (pending admin approval before access is granted) |
| `/dashboard` | Role-aware main dashboard — renders Admin view or Employee/Secretary view based on the logged-in user's role |
| `/dashboard/branch-manager` | Branch Manager dashboard — review and final approval of booking requests |
| `/dashboard/admin/users` | Admin user management — approve new registrations, view and manage all user accounts |
| `/dashboard/admin/settings` | Admin settings — manage rooms (lecture/multi-purpose), time slots, and semester fixed schedules |
| `/dashboard/admin/calendar` | Admin calendar — view all bookings across rooms in a calendar layout |
| `/dashboard/admin/search` | Admin search — query bookings and availability across rooms and dates |
| `/dashboard/admin/delegations` | Admin delegations — assign substitutes to cover employees during leave periods |
